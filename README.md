# Facial Emotion Recognition API

A deep learning-based REST API that detects faces and recognizes emotions from images in real time. Built with FastAPI and deployed using Docker.

## What it does
- Detects all faces in an uploaded image
- Predicts emotion for each detected face
- Returns face location, confidence score and emotion label
- Supports 7 emotions: Anger, Disgust, Fear, Happiness, Neutral, Sadness, Surprise

## Tech Stack
- [Python](https://python.org/)
- [PyTorch](https://pytorch.org/)
